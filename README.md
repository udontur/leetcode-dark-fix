# LeetCode Dark Fix
Adds Dark mode to LeetCode's home page, explore page, and settings page because LeetCode somehow doesn't have it (as of June 2025).

![](assets/demo.png)

## Installation
1. Install a userscript manager like [Tampermonkey](https://www.tampermonkey.net/).
2. Open the [GreasyFox](https://greasyfork.org/en/scripts/540164-leetcode-dark-fix) link.
3. Click the green button that says `Install this script`.
